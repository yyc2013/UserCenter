Oauth2.0接入说明



URL

HTTP请求方式

字段 必选 类型及范围 说明
client_id true string 申请应用时分配的App key
redirect_uri true string 授权回调地址，站外应用需与设置的回调地址一致，用于获取code。

返回数据

字段 字段类型 说明
code string 用于调用access_token，接口获取授权后的access token.


URL

HTTP请求方式

字段 必选 类型及范围 说明
client_id true string 申请应用时分配的App key
code true string 访问login通过登陆后返回的参数

返回数据

{"access_token":"ACCESS_TOKEN"}
